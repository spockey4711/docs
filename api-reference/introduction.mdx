## Authentication

Protected endpoints use Bearer token authentication.

```json
"security": [
  {
    "bearerAuth": []
  }
]
```

## Core flows

The current reference focuses on three key backend flows:

- user registration
- user login
- single-session fueling calculation

## Notes

The API should match the product boundary described in the product docs. Keep endpoints aligned with a narrow, deterministic calculator rather than broad coaching or wellness functionality.
