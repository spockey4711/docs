## Purpose

Fueling Optimizer gives endurance athletes precise fueling guidance for one training session.

The user provides a small set of inputs:

- body weight
- session duration
- session intensity
- sport type
- practical constraints such as carbohydrate tolerance or execution preferences

The system returns concrete targets across the full session timeline:

- before session
- during session
- after session

Each recommendation is expressed in operational units such as `g`, `ml`, and `mg`.

## Audience

The primary audience is ambitious endurance athletes who want precise and explainable guidance rather than vague coaching language.

This includes athletes training for long rides, long runs, bricks, race simulations, or structured sessions where fueling has a direct performance impact.

## Positioning

Fueling Optimizer is a sports science calculator.

It is not positioned as:

- an AI coach
- a recovery therapy product
- a long-term training planner
- an adaptive system that silently changes recommendations based on hidden logic

## Brand direction

The interface should feel clean, precise, athletic, and trustworthy.

- Primary brand colors: `#10b981`, `#059669`, `#14b8a6`
- Background direction: light green-tinted off-white `#f8faf9`
- Primary text: `#0f1a14`
- Secondary text: `#5f6e65`
- Semantic accents: blue for before-session, amber for during-session, purple for after-session
