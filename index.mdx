Fueling Optimizer helps ambitious endurance athletes turn session inputs into concrete fueling targets in `g`, `ml`, and `mg`.

It is intentionally narrow. The product covers one training session at a time and focuses on practical, explainable recommendations before, during, and after the session.

## Product posture

Fueling Optimizer is positioned as sports science software, not an AI coach.

<Note>
  Recommendations are deterministic, transparent, and intentionally free of black-box adaptation.
</Note>

## Instrumentation

PostHog can be enabled in both the shared SPA shell and the FastAPI backend using the same project key. The frontend uses the HTML snippet integration with a proxy `api_host` plus the PostHog EU `ui_host`. The backend uses the Python `posthog` package for successful registration, login, and calculation events.
