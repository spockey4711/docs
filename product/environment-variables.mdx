## Core runtime

```bash
DATABASE_URL=
LOGIC_VERSION=fueling-logic-v1
FUELING_LOGIC_MODE=matrix_v1
FUELING_LOGIC_SHADOW_COMPARE=false
APP_BASE_URL=http://localhost:8000
```

## Notes

- Without `DATABASE_URL`, calculation still works but persistence-backed features do not.
- Auth flows require DB persistence.
- Billing self-service requires DB state and Stripe configuration.
- PostHog is disabled when no API key is configured.
